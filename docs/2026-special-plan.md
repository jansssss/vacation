# 2026 노무 대응 특집

## 목표

- 기존 e-work.kr의 계산기, 가이드, 게시판 구조를 유지한다.
- 2026년 변경 이슈를 중심으로 검색 유입과 상담 전환을 강화한다.
- 대표 페이지는 `육아휴직/육아기 단축근무 지원금 계산기`로 잡는다.

## 범위

- 포함
- 2026 특집 메인 메시지 재구성
- 실수령액 계산기 2026 포지셔닝 강화
- 2026 최저임금 점검 계산기 추가
- 육아휴직/육아기 단축근무 지원금 계산기 추가
- 계산기별 가이드와 CTA 연결

- 제외
- AI 급여명세서 감사
- AI 사규 생성
- 파일 업로드 OCR
- LLM 연동 기능

## 1. 육아휴직/육아기 단축근무 지원금 계산기 PRD

### 역할

- 사업주가 받을 수 있는 지원금
- 근로자가 받을 수 있는 급여
- 회사의 실제 순부담 변화를 한 화면에서 계산한다.

### 핵심 사용자

- 10~50인 사업장 대표
- 인사/총무 실무자
- 노무 이슈를 직접 처리하는 중소기업 관리자

### 우선 지원 시나리오

1. 육아휴직 사용 시 사업주 지원금 계산
2. 육아기 근로시간 단축 사용 시 사업주 지원금 계산
3. 육아기 근로시간 단축 시 근로자 급여 추정
4. 업무분담 지원금, 인센티브, 계속고용 조건을 반영한 회사 순효과 계산

### 입력 구조

#### 공통 입력

- 제도 유형
- `육아휴직`
- `육아기 근로시간 단축`
- 자녀 개월 수 또는 연령 구간
- 사용 시작일
- 사용 개월 수
- 우선지원대상기업 여부
- 해당 근로자 월 통상임금
- 해당 근로자 월 기본급 또는 기준 급여
- 회사 내 허용 회차
- 최초 / 2회차 / 3회차 이상
- 복귀 후 6개월 계속고용 예정 여부

#### 육아휴직 추가 입력

- 연속 사용 여부
- 사용 개월 수가 3개월 이상인지
- 남성 육아휴직 인센티브 대상 여부
- 대체인력 사용 여부
- 업무분담 보상 지급 여부
- 사업장 규모
- `30인 미만`
- `30인 이상`

#### 육아기 근로시간 단축 추가 입력

- 주 소정근로시간
- 단축 후 주 근로시간
- 단축 전후 차감 시간
- 업무분담 보상 지급 여부
- 사업장 규모
- `30인 미만`
- `30인 이상`

### 출력 구조

#### 1차 요약 카드

- 근로자 월 예상 수급액
- 사업주 월 지원금
- 사용 기간 총 지원금
- 회사 월 순부담 변화

#### 상세 결과

- 근로자 기준
- 단축 전 임금
- 단축 후 회사 지급 임금
- 고용센터 급여 추정액
- 합산 월 수령액

- 사업주 기준
- 육아휴직 지원금
- 육아기 근로시간 단축 지원금
- 회차 인센티브
- 남성 육아휴직 인센티브
- 업무분담 지원금
- 대체인력 지원금은 후속 확장 대상으로 분리

- 회사 기준
- 기존 인건비
- 제도 사용 중 직접 인건비
- 정부지원금 차감 후 순부담
- 기존 대비 증감액

### 계산 정책

- 정책값은 하드코딩이 아니라 `2026 rules` 객체로 분리한다.
- 총요건 충족 전에는 경고와 참고값만 보여준다.
- 신청 가능성과 지급 확정액을 구분한다.
- `예상 지급액`과 `신청 가능 항목`을 분리해서 표시한다.

### 주의 문구

- 실제 지급 여부는 고용센터 심사 결과에 따라 달라질 수 있다.
- 우선지원대상기업 여부, 계속고용 요건, 월평균보수 등은 별도 확인이 필요하다.
- 업무분담 지원금은 동료 보상 지급 사실과 요건 충족이 필요하다.

### 페이지 구조

1. Hero
2. 빠른 요약 카드
3. 입력 패널
4. 결과 패널
5. 계산 기준 설명
6. 자주 틀리는 조건
7. 신청 체크리스트
8. 관련 가이드
9. 공지/게시판 연결

### 참고 정책 포인트

- Work24 사업주 지원금 안내에 따르면 육아휴직 지원금과 육아기 근로시간 단축 지원금은 우선지원대상기업 중심으로 운영된다.
- 육아휴직 지원금은 일반 월 30만원, 2026년부터 12개월 이내 자녀 연속 3개월 이상 사용 특례가 월 100만원으로 조정된다.
- 육아기 근로시간 단축 지원금은 월 30만원, 최초 3회 허용 사례까지 월 10만원 인센티브가 붙는다.
- 업무분담 지원금과 2026년 홍보자료 수치는 별도 제도 페이지 기준으로 rules에 분리 저장한다.
- 육아기 근로시간 단축 급여 상한 변경은 근로자 수급액 계산에 반영한다.

## 2. 메인페이지 IA

### 홈 역할

- 전체 계산기 허브 성격은 유지한다.
- 동시에 `2026 노무 대응 특집`의 대표 진입점을 맡는다.

### 우선순위

1. 육아휴직/육아기 단축근무 지원금 계산기
2. 2026 실수령액 계산기
3. 2026 최저임금 점검 계산기
4. 기존 계산기 허브

### 제안 구조

1. Hero
- 배지: `2026 노무 대응 특집`
- 제목: `바뀐 급여, 최저임금, 육아지원까지 계산기로 먼저 점검하세요`
- CTA 1: `육아지원금 계산기`
- CTA 2: `계산기 전체 보기`

2. 특집 핵심 카드 3개
- 육아지원금 계산기: 가장 크게 노출
- 실수령액 계산기
- 최저임금 점검 계산기

3. 2026 체크포인트
- 최저임금
- 4대보험/급여 반영
- 육아기 근로시간 단축
- 퇴직/연금 체크

4. 기존 계산기 목록

5. 가이드/게시판 업데이트

### 홈에서의 메시지 원칙

- 법률 공포보다 실무 편의에 집중한다.
- `복잡한 숫자를 대신 정리한다`는 인상을 준다.
- 대표 CTA는 항상 육아지원금 계산기로 보낸다.

## 3. 구현 시작 범위

### 1차 구현

- 새 계산기 라우트 추가
- 계산기 레지스트리에 육아지원금 계산기 추가
- 홈에 2026 특집 섹션 추가
- 계산기 허브 상단에 특집 진입 카드 추가
- rules 파일에 육아지원금용 2026 정책값 초안 추가
- 계산기 엔진과 테스트 추가

### 2차 구현

- 최저임금 계산기 추가
- 육아지원금 관련 가이드 작성
- 랜딩형 세부 페이지 연결

## 확인할 공식 출처

- 고용24 사업주 지원 제도 안내
- 고용노동부 상담/홍보자료의 2026 일가정양립 지원 변경사항
- 육아기 근로시간 단축 급여 상한 관련 고용노동부 안내
